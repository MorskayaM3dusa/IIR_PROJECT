# Система подсчёта и классификации объектов по цвету

**Компьютерное зрение | Подсчёт объектов | Классификация по цвету**


## Описание проекта

Проект реализует систему для подсчёта и классификации движущихся объектов по цвету в реальном времени. Используются модули `CV_2_04` и `CV_2_43` для обработки видео, обнаружения объектов и визуализации результатов. Статистика подсчёта сохраняется в текстовый файл.


## Возможности

- **Захват видео**: Работа с веб-камерой или видеофайлами.
- **Классификация по цвету**: Поддержка 5 цветов: красный, синий, зелёный, жёлтый, оранжевый.
- **Подсчёт объектов**: Фильтрация объектов по площади (минимальная и максимальная).
- **Визуализация**: Отображение контуров, меток и счётчиков на кадре в реальном времени.
- **Статистика**: Сохранение детальной статистики (среднее, максимум, минимум, стандартное отклонение) в файл.
- **Настройка параметров**: Возможность задавать минимальную/максимальную площадь объектов, коэффициент изменения размера кадра, имя выходного файла.

## Требования

- Python 3.8+
- Библиотеки:
  - `opencv-python`
  - `numpy`

### Установка

Клонируйте репозиторий или скачайте файлы проекта.

`python main.py --source video.mp4 --min_area 1000 --max_area 20000 --output processed.mp4 --stats stats.txt`

# Проект: Биннинг числовых признаков

## Описание проекта
Этот проект демонстрирует метод биннинга (разделения на интервалы) числовых признаков. Проект включает загрузку данных (встроенный датасет diabetes или пользовательский CSV файл), разделение числового признака на интервалы и визуализацию результатов.

## Возможности
- **Загрузка данных**: Использование встроенного датасета diabetes или загрузка пользовательского CSV файла
- **Биннинг признаков**: Разделение числового признака на заданное количество интервалов
- **Стратегии биннинга**: Поддержка равных интервалов и квантильного разделения
- **Визуализация**: Построение bar plot для анализа распределения по категориям
- **Оценка качества**: Расчет метрик равномерности распределения
- **Сохранение результатов**: Экспорт обработанных данных в CSV файл
